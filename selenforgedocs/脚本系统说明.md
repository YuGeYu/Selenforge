# Selenforge 脚本系统说明

## 概览

Selenforge 当前的主脚本系统是桌面端内置的 MoonLang 脚本链路，不直接依赖 PlayCanvas Editor 原生的 JS Script Asset 解析流程。

脚本系统现在包含三层：

1. 编辑层
   在桌面编辑器中创建、编辑、复制、删除 Moon 脚本。
2. 编译层
   `desktop/moon-script-tools.js` 负责把 MoonLang 子集编译为可预览的 JavaScript。
3. 运行层
   `desktop/game-template.js` 负责在游戏运行时装载编译结果，并提供一组 Moon 风格内置函数。

## 当前增强点

本次完善主要补了四部分：

1. 脚本编辑器增强
   支持脚本名、脚本描述编辑。
   支持模板套用。
   支持脚本复制。
   支持脚本删除。
   不使用浏览器原生弹窗，全部通过编辑器面板和项目内确认框完成。

2. Moon 编译器增强
   支持 `function / if / elif / else / while / for in range / for = to / try / catch / throw / moon / return / break / continue / global`。
   对未支持的 `class`、`switch` 等结构给出明确编译错误。
   编译错误会带行号，例如 `line 8: missing end for if block`。

3. 运行时内置函数增强
   游戏运行时和桌面预览运行时都补充了常用 Moon 风格内置函数：
   `print`
   `type`
   `str / int / float / bool`
   `len`
   `append / insert / pop / remove`
   `contains / index_of`
   `join / split / trim / to_upper / to_lower / substring / starts_with / ends_with / replace`
   `abs / min / max / floor / ceil / round / sqrt / pow / random / random_int`
   `now / sleep`

4. 模板增强
   当前内置三个模板：
   `game`
   `rotator`
   `system`

## 脚本角色

### `game`

用于游戏运行时脚本。

这类脚本会：

1. 保存到快照目录的 `scripts/*.moon`
2. 生成对应的 `runtime/*.js`
3. 在游戏构建产物里被自动加载

推荐入口函数：

```moonlang
function start():
    print("game script start")
end

function update(dt):
    return dt
end
```

### `system`

用于桌面壳侧执行的系统脚本。

这类脚本可以在脚本编辑器里点击“运行系统脚本”，由：

1. `main.moon`
2. `MulanMoonlang/moonc.exe`

完成临时编译与执行。

适合做：

- 批处理任务
- 构建辅助
- 本地工具脚本

## 模板说明

### `Game Loop`

默认游戏脚本模板，适合普通逻辑入口。

### `Rotator`

用于快速演示对象旋转。
默认会尝试查找名为 `月铸核心` 的实体并持续旋转。

### `System Task`

用于系统脚本执行，默认包含 `start()` 入口。

## 推荐工作流

1. 在资源面板中新建 Moon 脚本。
2. 在脚本编辑器中修改名称、描述和角色。
3. 根据需要套用模板。
4. 观察右侧 JS 预览与错误信息。
5. 点击“应用到项目”。
6. 保存项目。
7. 需要桌面侧执行时，把角色切换为 `system` 再运行。
8. 需要打包时执行构建。

## 语法边界

当前编译器是 MoonLang 子集编译器，不是完整 MoonLang 编译器。

已支持：

- 常见控制流
- 基本函数定义
- 常用表达式
- 常用内置函数

暂未在前端子集编译器中支持：

- `class`
- `switch`
- 完整模块系统
- 完整标准库
- 复杂协程调度

如果脚本需要完整 MoonLang 能力，优先使用：

- 系统脚本执行链
- 正式的 `moonc.exe` 编译链

## 关键文件

- `desktop/app.js`
  脚本编辑器 UI、脚本管理、项目内脚本操作
- `desktop/moon-script-tools.js`
  MoonLang 子集编译器与桌面运行时内置函数
- `desktop/game-template.js`
  游戏运行时脚本装载与内置函数桥接
- `main.moon`
  系统脚本执行、快照保存、游戏构建
